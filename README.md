# fetch-api
fetching API
Choosed a free public API quotes.
Created a simple Ul with input, button, and result display section.
Used JavaScript function using the fetch() API to send HTTP requests to the API endpoint.
Handled API responses by converting them into JSON format and extracting required fields.
Display fetched data dynamically on the web page using DOM manipulation.
Implemented error handling using.catch() and conditional checks for failed responses.
Added loading indicators to inform users while data is being fetched.
Test API behavior under network failure scenarios using browser DevTools.
